Проект ученика 9-3 класса Державинского лицея

На этой странице я хотели бы рассказать про себя (создателя) и "технологию" создания данного сайта.

Меня зовут Барсуков Святослав, я ученик 9-3 класса Державинского лицей (на момент 2022-2023 годов).
Я увлекаюсь программированием, веб дизайном и разработкой сайтов, по профессии хочу пойти на full-stack разработчика, чтобы создавать крутые сайты для разныйх компаний :)
Выбор темы для проекта был очевиден, я давно хотел попробовать написать сайт только на HTML, при этом чтобы он был функциональный и "красивый".

В моих планах было создание шаблона полноценного функционального сайта, при этом чтобы он умещался в одной файле (чтобы другим людям было проще разобраться), поэтому использовать CSS и JavaScript было нельзя, и казалось, что без них сверстать нормальный сайт невозможно.
Но, как оказалось, возможно сверстать нормальный сайт и при помощи одного только HTML.
Раньше (в 2000х годах), когда CSS не было, сайты верстали на таблицах, поэтому решение проблемы оказалось довольно простым - попробовать воссоздать ту технологию и сверстать сайт при помощи таблиц.
Получилось или нет - решать вам :)
